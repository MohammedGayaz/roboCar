
# Robot car

A Simple Car that is controlled by Arduino Uno and l293D motor driver.

## Functionality
- Follow line
- Avoid Obstacle




## Components
- Arduino Uno
- l293D driver shield
- Bo motors (x4)
- IR sensors (x2)
- 4WD Car Chassis
- Joy stick (Optinal)
- 2 Pin SPST on-off-on Switch
- 9V Battry
- Some Jumper Wires
